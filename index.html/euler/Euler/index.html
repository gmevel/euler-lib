Module Euler

This is a library for arithmetic algorithms, primarily developed to solve Project Euler problems, although it is of more general use.

val pow : mult:('a -> 'a -> 'a) -> unit:'a -> 'a -> int -> 'a

Generic fast exponentiation. pow ~mult ~unit x n is unit composed n times to the right with x, provided that n is non‚Äênegative. For example:

yields "xyyyyy". Complexity: ùí™(log(n)) calls to mult.

  • parameter unit

    the left‚Äêmost operand of the product (most often, we use a neutral element for mult).

val memoized_fix : (('a -> 'b) -> 'a -> 'b) -> 'a -> 'b

Memoizing fixpoint combinator. Example use:

let fib = memoized_fix@@fun fib n ->
  if n < 2 then
    1
  else
    fib (n-1) + fib (n-2)
